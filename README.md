# ZKTECO Cihazları Kullanım Dökümanı

Merhaba,

Bu repoda, ZKTECO gibi parmak izli ve yüz tanımalı cihazlarda personel takibi için cihaz protokollerini kullanarak nasıl işlemler yapıldığını gösterdim. PHP, JS ve Bootstrap kullanarak geliştirdiğim bu yazılımda aşağıdaki modülleri kullanabilirsiniz:

- **Ses Testi**: Cihazın sesini test edebilirsiniz.
- **Cihaz Adı Değiştirme**: Cihazın adını değiştirebilirsiniz.
- **Kayıtlı Kullanıcıları Görme**: Cihazda kayıtlı kullanıcıları görebilirsiniz.
- **Yeni Kullanıcı Kayıt Etme**: Yeni kullanıcılar ekleyebilirsiniz.
- **Kullanıcı Düzenleme/Silme**: Mevcut kullanıcıları düzenleyebilir veya silebilirsiniz.
- **Giriş Çıkış Verileri**: Kullanıcıların giriş çıkış verilerini görebilirsiniz.
- **Giriş Çıkış Verilerini Temizleme**: Giriş çıkış verilerini temizleyebilirsiniz.
- **Cihaz Saatini Değiştirme**: Cihazın saatini değiştirebilirsiniz.

Sistem entegrasyonunuz için ya da daha gelişmiş özellikler isterseniz iletişime geçebilirsiniz.
